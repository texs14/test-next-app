# Авторизация отключена

## Что было сделано

Авторизация в приложении была отключена путем комментирования соответствующего кода. Код не был удален, а только закомментирован для возможности быстрого восстановления.

## Измененные файлы

### 1. `src/app/layout.tsx`
- Закомментированы `<AuthProvider>` и `<ProtectedLayout>`
- Приложение теперь работает без проверки авторизации

### 2. `src/components/ProtectedLayout.tsx`
- Закомментирована вся логика проверки авторизации
- Компонент теперь просто возвращает дочерние элементы

### 3. `src/components/Nav.tsx`
- Закомментирован `useAuth()` хук
- Закомментирована проверка пользователя
- Закомментирован `<UserMenu>`

### 4. `src/app/settings/page.tsx`
- Закомментирован код работы с авторизацией
- Добавлено сообщение "Авторизация отключена" при попытке сохранения

### 5. `src/app/profile/page.tsx`
- Закомментирован код загрузки данных пользователя
- Добавлены заглушечные данные для демонстрации

## Как восстановить авторизацию

Чтобы восстановить авторизацию, необходимо:

1. Раскомментировать все закомментированный код во всех перечисленных файлах  
2. Убрать заглушечные данные в `profile/page.tsx`
3. Убрать сообщение "Авторизация отключена" в `settings/page.tsx`
